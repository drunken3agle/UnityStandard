## Tips/Tricks/Hacks

### Getting Unity dark mode

1. Install HxD
2. Run in admin-mode and open Unity.exe
3. Search for `84 C0 75 08 33 C0 48 83 C4 20 5B C3 8B 03 48 83 C4 20 5B C3` in hex mode
4. Replace `75` with `74`

Tested with Unity 2018.2.7f1
